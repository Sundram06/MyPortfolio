# Portfolio - Sundram Gupta

A modern, responsive portfolio website showcasing my work as a Full-Stack Software Engineer.

## 🚀 Tech Stack

- **Frontend**: React 18, TypeScript
- **Styling**: Tailwind CSS with custom design tokens
- **Build Tool**: Vite
- **Icons**: React Icons
- **Deployment**: Vercel

## ✨ Features

- Responsive design across all device sizes
- Light/Dark mode support with centralized theme system
- Featured project case study (VittNest)
- Work experience timeline
- Technical skills showcase
- Contact section with professional links

## 🛠️ Development

### Prerequisites

- Node.js 18+ and npm

### Installation

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

## 📦 Project Structure

```
portfolio1/
├── src/
│   ├── components/     # Reusable components
│   ├── sections/       # Page sections (Hero, Project, Experience, etc.)
│   ├── lib/            # Utilities and theme logic
│   ├── globals.css     # Theme tokens and global styles
│   ├── App.tsx         # Main app component
│   └── main.tsx        # Entry point
├── public/             # Static assets
└── dist/               # Production build output
```

## 🌐 Deployment

This project is optimized for Vercel deployment:

1. Push your code to GitHub
2. Import the repository in Vercel
3. Vercel auto-detects Vite configuration
4. Deploy with default settings
